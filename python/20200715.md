```javascript
a - b
a * b

```

문자열
```javascript

'Hello world'
"""Hello world"""

```

문자열 연산
```javascript
head = "Python"
tail = "is fun!"
head + tail

결과
Python is fun!

head * 2

결과
PythonPython
```

문자열 길이 
```javascript
a = "Life is too short"
len(a)

결과
17
```

문자열 슬라이싱
```javascript
a = "Life is too short, You need Python"
a[0:4]

결과
Life

a[:4] -> 처음부터 a[3] 까지
a[4:8] -> a[4] 부터 a[7] 까지
a[8:] -> a[8] 부터 마지막 까지
```
