# 파이썬 기초 문법1

조건문 if
```javascript
a = 3
if a > 1:
    print("a is greater than 1")

결과
a is greater than 1
```

반복문 for
```javascript
for a in [1,2,3]:
    print(a)

결과 
1
2
3
```

반복문 while
```javascript
i = 0
while i < 3:
    i = i + 1
    print(i)

결과
1
2
3
```

함수
```javascript
def add(a,b):
    return a + b 

add(3,4)

결과
7
```
